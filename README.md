# Simple-DiscordBOT
Simi DiscordBOT by DrelezTM

## Installation
```bash
> git clone https://github.com/DrelezTM/SimiBOT-Discordjs
> cd Simple-DiscordBOT
> Fill in the CHANNEL ID in config.json
> Fill in the BOT TOKEN in config.json
> bash install.sh
```
